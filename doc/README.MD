
# PokeCreate

## Estrutura do Projeto

```
POKECREATE (root)
└── PokeCreate
    └── PokeCreate
        └── CORE
            ├── Classes
            │   ├── Pokemon
            │   └── Usuario
```

## Descrição

**PokeCreate** é um projeto estruturado em camadas com o seguinte layout:

- **CORE**: Núcleo da aplicação contendo as classes principais
  - **Pokemon**: Classe responsável pela modelagem e manipulação de Pokémon
  - **Usuario**: Classe responsável pela modelagem e manipulação de usuários

## Como Usar

Navegue até a pasta `CORE` para acessar as classes principais do projeto.
